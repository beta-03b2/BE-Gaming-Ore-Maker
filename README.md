## English

A simple tool to make blocks like ores "Gaming Style" in Minecraft: BE.

[Open the Site](https://beta-03b2.github.io/BE-Gaming-Ore-Maker)

- Automatically generates resource packs with rainbow animations.
- No need to touch complicated JSON files.
- Adjust transparency and speed on the web, then download.
